# Iso646.h

The header defines alternative spellings for several operators to improve code readability and compatibility.

### Alternative Operator Spellings

This header provides macro definitions that map alternative spellings to standard operators, allowing more readable code or compatibility with certain coding standards.

#### Defined Macros

- `and` → `&&`: Logical AND operator
- `and_eq` → `&=`: Bitwise AND assignment
- `bitand` → `&`: Bitwise AND operator
- `bitor` → `|`: Bitwise OR operator
- `compl` → `~`: Bitwise complement operator
- `not` → `!`: Logical NOT operator
- `not_eq` → `!=`: Not equal operator
- `or` → `||`: Logical OR operator
- `or_eq` → `|=`: Bitwise OR assignment
- `xor` → `^`: Bitwise XOR operator
- `xor_eq` → `^=`: Bitwise XOR assignment

### Example Usage

This example demonstrates the use of alternative operator spellings to improve readability in C code.

```c
#include <iso646.h>
#include <stdio.h>

int main() {
    int a = 1, b = 0;
    if (a and not b) {
        printf("Alternative operators work!\n");
    }
    return 0;
}
```
